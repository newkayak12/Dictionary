# 정의
- 문자열 집합을 트리 구조로 저장하는 자료구조

# 핵심 아이디어
- 공통 접두사를 가진 문자열들이 경로를 공유
- 각 노드는 하나의 문자를 나타냄
- 루트에서 리프까지의 경로 = 하나의 문자열

# 기본 구조
```text
문자열 집합: ["CAR", "CARD", "CARE", "CAT"]

        ROOT
         |
         C
         |
         A
        / \
       R   T
      /|\   \
     D E $   $
     | |
     $ $
```

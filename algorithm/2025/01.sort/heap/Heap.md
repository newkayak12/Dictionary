- 힙은 완전 이진 트리 형태로 최대, 최소 값을 빠르게 찾아내는 유용한 자료 구조
- 중복 값을 허용한다.
- 부모-자식 간 정렬은 보장하고, 형제간의 정렬은 보장하지 않는다. -> 반 정렬 상태
- 최대 힙, 최소 힙이 있다.
	- 최소 힙 -> 루트 노드가 최솟 값이 되고, 부모 노드의 key는 자식 노드의 key보다 작아야 한다.
	- 최대 힙 -> 루트 노드가 최댓 값이 되고, 부모 노드의 key가 자식 노드의 key보다 커야 한다.

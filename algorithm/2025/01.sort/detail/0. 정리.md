1. 버블
	1. 인접 두 개씩 비교
	2. 한 바퀴 돌면 최대 값이 맨 뒤
	3. 회전 당 맨 뒤 인덱스는 하나씩 배제
2. 선택
	1. 현재 인덱스까지 포함해서 앞쪽을 정렬한다.
	2. 해당 구간 내 최소 값을 찾고 그 값은 현재 인덱스와 스왑
	3. 회전 당 맨 앞 인덱스 하나씩 배제
3. 삽입
	1. 현재 원소 앞에 있는 값 중 현재 원소보다 크다면 하나씩 민다
	2. 현재 원소보다 작은 수가 나올 때까지 밀고 이전 인덱스에 삽입
4. 셸
	1. Gap 산정 (보통 array / 2로 시작)
	2. Gap 간 요소 삽입 정렬 진행한다 (Gap 간 하나씩 민다.)
5. 병합
	1. 원소가 하나 남을 때까지 배열을 분할한다.
	2. 합치면서 정렬한다.
		1. 합칠 사이즈의 배열을 지정한다.
		2. 두 개의 배열을 비교하면서 새 배열 앞부터 채운다.
		3. 남은 배열들은 각 값 (mid, end)와 비교하면서 하나씩 채운다.
	3. 반복 (분할 정복)
6. 퀵
	1. 중간 인덱스 값으로 피벗잡고 왼쪽, 오른쪽 투포인터가 피벗쪽으로 이동
	2. 왼쪽은 피벗보다 큰 값, 오른쪽은 작은 값의 인덱스를 맞바꾼다.
	3. 왼쪽 인덱스가 오른쪽보다 작거나 같은 경우까지 진행
	4. 피벗은 고정하고 좌우 파티셔닝해서 다시 진행한다.

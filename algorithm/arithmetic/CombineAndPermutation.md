# Combine

서로 다른 `n`개의 물건에서 순서를 생각하지 않고 `r`개를 택할 때, 이것은 `n`개에서  `r`개를 택하는 조합이라고 하고 이 조합의 수를 기호로 
<sub>n</sub>C<sub>r</sub>이라고 한다.

ex)
input: [1, 2, 3, 4]
output: [ [1, 2, 3], [1, 2, 4], [1, 3, 4], [2, 3, 4] ]

즉, [1, 2, 3], [2, 3, 1] 순서는 상관이 없다는 것을 의미한다.

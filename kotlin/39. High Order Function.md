## **1. 변환 (Transformation)**

### **map** - 각 요소 변환

kotlin

```kotlin
listOf(1, 2, 3).map { it * 2 }
// [2, 4, 6]
```

### **mapIndexed** - 인덱스 포함 변환

kotlin

```kotlin
listOf("a", "b").mapIndexed { idx, value -> "$idx: $value" }
// ["0: a", "1: b"]
```

### **flatMap** - 중첩 컬렉션 평탄화

kotlin

```kotlin
listOf(1, 2).flatMap { listOf(it, it * 2) }
// [1, 2, 2, 4]
```

---

## **2. 필터링 (Filtering)**

### **filter** - 조건 만족 요소만

kotlin

```kotlin
listOf(1, 2, 3, 4).filter { it % 2 == 0 }
// [2, 4]
```

### **filterNot** - 조건 불만족 요소만

kotlin

```kotlin
listOf(1, 2, 3).filterNot { it > 2 }
// [1, 2]
```

### **filterNotNull** - null 제외

kotlin

```kotlin
listOf(1, null, 3).filterNotNull()
// [1, 3]
```

### **take / drop**

kotlin

```kotlin
listOf(1, 2, 3, 4).take(2)      // [1, 2]
listOf(1, 2, 3, 4).drop(2)      // [3, 4]
listOf(1, 2, 3, 4).takeLast(2)  // [3, 4]
```

---

## **3. 집계 (Aggregation)**

### **reduce** - 누적 연산 (초기값 없음)

kotlin

```kotlin
listOf(1, 2, 3).reduce { acc, value -> acc + value }
// 6 (1+2=3, 3+3=6)
```

### **fold** - 누적 연산 (초기값 있음)

kotlin

```kotlin
listOf(1, 2, 3).fold(10) { acc, value -> acc + value }
// 16 (10+1+2+3)
```

### **sum / count / max / min**

kotlin

```kotlin
listOf(1, 2, 3).sum()           // 6
listOf(1, 2, 3).count()         // 3
listOf(1, 2, 3).maxOrNull()     // 3
```

---

## **4. 그룹화 (Grouping)**

### **groupBy** - 조건별 그룹화

kotlin

```kotlin
listOf(1, 2, 3, 4).groupBy { it % 2 }
// {1=[1,3], 0=[2,4]}
```

### **partition** - true/false로 분리

kotlin

```kotlin
listOf(1, 2, 3, 4).partition { it % 2 == 0 }
// ([2, 4], [1, 3])
```

### **chunked** - 고정 크기로 분할

kotlin

```kotlin
listOf(1, 2, 3, 4, 5).chunked(2)
// [[1,2], [3,4], [5]]
```

---

## **5. 연관 (Association)**

### **associateWith** - 요소를 Key로

kotlin

```kotlin
listOf("a", "b").associateWith { it.length }
// {a=1, b=1}
```

### **associateBy** - 람다 결과를 Key로

kotlin

```kotlin
listOf("abc", "de").associateBy { it.length }
// {3=abc, 2=de}
```

### **associate** - Key-Value 모두 지정

kotlin

```kotlin
listOf(1, 2).associate { it to it * 2 }
// {1=2, 2=4}
```

## **6. 검사 (Checking)**

### **any / all / none**

kotlin

```kotlin
listOf(1, 2, 3).any { it > 2 }      // true
listOf(1, 2, 3).all { it > 0 }      // true
listOf(1, 2, 3).none { it < 0 }     // true
```

### **contains / containsAll**

kotlin

```kotlin
listOf(1, 2, 3).contains(2)                 // true
listOf(1, 2, 3).containsAll(listOf(1, 3))   // true
```

## **7. 정렬 (Sorting)**

### **sorted / sortedBy / sortedDescending**

kotlin

```kotlin
listOf(3, 1, 2).sorted()                    // [1, 2, 3]
listOf("ab", "a").sortedBy { it.length }    // [a, ab]
listOf(1, 2, 3).sortedDescending()          // [3, 2, 1]
```

---

## **8. 중복 제거 (Deduplication)**

### **distinct / distinctBy**

kotlin

```kotlin
listOf(1, 2, 2, 3).distinct()               // [1, 2, 3]
listOf("a", "ab", "b").distinctBy { it.length }  // [a, ab]
```

## **9. 조합 (Combination)**

### **zip** - 두 컬렉션 결합

kotlin

```kotlin
listOf(1, 2).zip(listOf("a", "b"))
// [(1, a), (2, b)]
```

### **zipWithNext** - 인접 요소 결합

kotlin

```kotlin
listOf(1, 2, 3).zipWithNext()
// [(1, 2), (2, 3)]
```

---

## **10. 실행 (Action)**

### **forEach / forEachIndexed**

kotlin

```kotlin
listOf(1, 2).forEach { println(it) }
listOf(1, 2).forEachIndexed { idx, value -> println("$idx: $value") }
```

### **onEach** - 체이닝용 forEach

kotlin

```kotlin
listOf(1, 2, 3)
    .onEach { println(it) }
    .map { it * 2 }
// 출력 후 [2, 4, 6] 반환
```
